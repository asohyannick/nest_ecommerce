# E-commerce Backend API

**Author:** Asoh Yannick  
**Role:** Backend Software Engineer  

This is the backend of my latest **e-commerce RESTful API**, developed using **Node.js**, **Nest.js**, and **MongoDB**. I am proficient in modern backend technologies including **TypeScript**, **Node.js**, **Nest.js**, **Git**, and **MongoDB**.  

This project implements the following **business requirements**:

- ✅ **[✔]** User authentication and role-based access control mechanism for **Admin**, **Super Admin**, and **Customers**  
- ✅ **[✔]** User registration and login API endpoints have been developed successfully  
- ✅ **[✔]** Reset and forgot password endpoints implemented successfully  
- ✅ **[✔]** Update and account deletion endpoints working successfully  
- ✅ **[✔]** Fetching all users by Admin or Super Admin successfully  
- ✅ **[✔]** Creation of products successfully by Admin or Super Admin  
- ✅ **[✔]** Advanced search functionalities such as sorting, filtering, pagination, and searching products by name, price, category, and more  
- ✅ **[✔]** Fetching all available products from the database by the Admins successfully  
- ✅ **[✔]** Deletion of products from the database by Admin or Super Admin successfully  
- ✅ **[✔]** Fetching a single product from the database successfully by Admin or Super Admin  
- ✅ **[✔]** Integration with a third-party payment gateway powered by **Stripe API** to accept payments from customers  
- ✅ **[✔]** Admin can block and unblock the account of customers successfully  
- ✅ **[✔]** User and Admin logout endpoints implemented successfully  
- ✅ **[✔]** And much, much more…
